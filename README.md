# Excel Sales Data Analysis Project

## Description
Analyzed sales data of 500 records using Microsoft Excel to generate business insights.

## Key Work Done
- Created Pivot Tables for region-wise and product-wise sales
- Grouped products and sales ranges for better analysis
- Created calculated fields to compute profit
- Added slicers for interactive filtering
- Designed bar and pie charts for visualization

## Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated Fields
- Slicers
- Charts
